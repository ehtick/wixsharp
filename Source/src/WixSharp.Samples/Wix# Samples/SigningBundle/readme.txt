This sample demonstrates how to digitally sign the produced EXE bootstrapper.If you're not planning to ship the MSI alone, there is no need to sign the inner MSI and cabs when signing a bundle. WiX has its own integrity verification system.Learn more: https://wixtoolset.org/docs/tools/signing/Execute corresponding .cmd file to build desired msi.Please note that the temporary certificate file (wixsharp.pfx) is for demonstration purposes only and it is not expected to lead to succesfull signing.
